# 👑 One Lord - Estilo e Autoridade

Bem-vindo ao repositório oficial da **One Lord**, uma grife focada em vestuário premium com propósito e design minimalista.

## 🚀 Sobre o Projeto
Este é o site oficial da marca, desenvolvido para oferecer uma experiência de compra rápida e direta via WhatsApp. O foco é a simplicidade e a elegância, destacando a qualidade das nossas peças.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura do site.
* **CSS3**: Estilização moderna e responsiva (adaptada para celulares).
* **GitHub Pages**: Hospedagem gratuita do site.
